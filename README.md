# Polykai

> A dark color scheme for code editors inspired by Wimer Hazenberg's [Monokai](https://github.com/monokai) theme.

![screenshot](https://raw.githubusercontent.com/adamgraham/polykai/master/images/screenshot.png)

## Getting Started
